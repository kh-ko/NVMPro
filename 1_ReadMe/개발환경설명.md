
## 1. 가상환경 생성 (Python 3.12)
프로젝트 루트 폴더(예: `NVMPro`)에서 아래 명령어를 실행하여 `venv`라는 이름의 가상환경을 생성합니다. 
(Python 3.12 버전이 시스템에 설치되어 있어야 합니다.)

```bash
# Python 3.12 환경을 명시적으로 사용할 경우
python3.12 -m venv venv
```

## 2. 가상환경 실행 (활성화)
**명령 프롬프트(CMD):**
```cmd
.venv\Scripts\activate.bat
```
**PowerShell:**
```powershell
.venv\Scripts\Activate.ps1
```
> **참고:** 정상적으로 활성화되면 터미널 좌측에 `(venv)`라는 접두사가 표시됩니다.

## 3. pip 최신 버전 업데이트
먼저 패키지 관리자인 `pip`를 최신 버전으로 업데이트합니다.
```bash
python -m pip install --upgrade pip
```

## 4. 패키지 설치
특정 버전의 `PySide6`와 `pyqtdarktheme-fork`을 수동으로 설치해야 한다면, 아래 명령어를 차례대로 입력하세요.
```bash
pip install PySide6==6.10.2
pip install pyqtgraph==0.14.0
pip install pyserial==3.5
pip install pyqtdarktheme-fork==2.3.4
pip install pydantic
```
