## 1. 가상환경 실행 (활성화)
**명령 프롬프트(CMD):**
```cmd
.venv\Scripts\activate.bat
```
**PowerShell:**
```powershell
.venv\Scripts\Activate.ps1
```
> **참고:** 정상적으로 활성화되면 터미널 좌측에 `(venv)`라는 접두사가 표시됩니다.

## 2. 리소스파일 만들기
PySide6 패키지에 내장된 pyside6-rcc 도구를 이용해 리소스 파일을 파이썬 py 파일로 변환합니다. 터미널(가상환경 상태) 에서 다음 명령어를 실행해 주세요.
```bash
.\.venv\Scripts\pyside6-rcc resources.qrc -o resources_rc.py
```

## 3. 앱 실행
패키지가 모두 설치되었다면, 아래 명령어로 본 애플리케이션을 실행합니다.
```bash
python main.py
